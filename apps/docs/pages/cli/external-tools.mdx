# Working with external tools

`run_security_tool(session_id, tool, args, timeout=120)` executes any
allow-listed tool via safe `subprocess` (no shell). Pencheff integrates
auxiliary tools across every phase of a pentest. Core port mapping, SQLi,
web exposure, and template detection are handled by Pencheff first-party
engines.

## Invocation

```text
pencheff sqli --url "https://target/login?user=test" --profile deep \
  --risk 2 --level 4 --traffic-log .pencheff/sqli-evidence.jsonl
```

Pencheff only executes tools from the allow-list in
[`server.py`](https://github.com/BalaSriharsha-Ch/pencheff/blob/main/plugins/pencheff/pencheff/server.py).
There is no shell interpolation — `args` is passed as a list directly
to `subprocess`.

## Which tools are installed?
